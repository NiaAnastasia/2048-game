# 2048 Game 🎮

A browser-based implementation of the classic 2048 puzzle game, 
built with vanilla JavaScript using OOP principles.

## Demo
🔗 [Live Preview](https://niaanastasia.github.io/2048-game/)

## How to Play
- Press **Start** to begin
- Use **arrow keys** (desktop) or **swipe** (mobile) to move tiles
- Tiles with the same number merge when they collide
- Reach **2048** to win — or keep going for a higher score!

## Technologies Used
- HTML5, SCSS
- JavaScript ES6+ (OOP, Classes)
- Gulp for build (SCSS compilation, dev server)
- ESLint + Stylelint for code quality
- GitHub Pages for deployment

## Getting Started

1. Clone the repo
```
git clone https://github.com/NiaAnastasia/2048-game.git
cd 2048-game
```

2. Install dependencies
```
npm install
```

3. Run the project locally
```
npm start
```

## Features
- 🎮 Smooth tile movement and merging
- 🏆 Score tracking
- 🎉 Win / 💀 Lose state messages
- 🔄 Restart button
- 📱 Swipe support for mobile
- 🎨 Color-coded tiles via SCSS modifiers
- ♟️ Game logic built with OOP (Game class)
